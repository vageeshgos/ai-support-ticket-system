# 🎫 AI Support Ticket System

An **AI-powered support ticket system** that automatically classifies, prioritizes, and routes incoming support tickets using NLP and LLM.

---

## 📌 What is this project?

This system takes raw customer support tickets, uses AI to understand the issue, classify its category and urgency, and auto-routes it to the right team. Built with Streamlit for the UI and LangChain for AI processing.

---

## 🗂️ Project Structure

| File | Description |
|---|---|
| `streamlit_app.py` | Main Streamlit web application |
| `requirements.txt` | Python dependencies |

---

## ✨ Features

- Auto-classify ticket category (Billing, Technical, General)
- AI-generated priority level (High, Medium, Low)
- Smart routing to the correct department
- Clean Streamlit dashboard UI
- LLM-powered response suggestions

---

## 🛠️ Tech Stack

- Python
- LangChain
- OpenAI / Groq LLM
- Streamlit
- NLP Processing

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
